Step 4: Add a cover image

  1. Choose a cover image. (Any jpg image can be used)
  2. Resize image to shrink the longest side down to 500px
  3. Renamed the shrinked image to "cover.jpg" and copy the file
  4. Navigate to the album folder and paste the cover image
