Step 3: Move Images

  1. Copy the thumbnails
  2. From the site root (docs folder) Navigate to assets -> gallery -> uploads
  3. Create a folder with the album id exactly as in the album file you downloaded in step 1
  4. Inside the folder you just created, create another folder named "thumbs"
  5. Paste the thumnail images you copied into this folder
  6. Now copy the images resized to 1920px (in step2, part 3)
  7. Navigate back to the album folder
  8. create a folder named "full-size" and paste the copied images into it.
